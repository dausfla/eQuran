* **Daftar Surah Lengkap**: Menampilkan 114 surah.

* **Navigasi & UI/UX**: Antarmuka pengguna yang responsif (cocok untuk mobile dan desktop) dan mudah digunakan.
* **Pencarian Cepat**: Fitur pencarian untuk memfilter daftar surah berdasarkan nama atau terjemahan.
* **Mode Gelap (Dark Mode)**: Pilihan tampilan mode gelap yang disimpan menggunakan *Local Storage*.
* **Handling Basmalah**: Penanganan otomatis untuk menampilkan `Basmalah` di awal surah, kecuali Surah Al-Fatihah (1) dan At-Taubah (9).
